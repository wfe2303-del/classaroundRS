# 수정내역

## 이번 수정 요약
- 로그인 화면을 **로고 + 버튼 1개** 구조로 단순화했습니다.
- 상단 제목 아래 노출되던 **시트 링크 문구를 제거**했습니다.
- **결제자 동기화 버튼을 제거**하고, 시트 탭을 선택하면 **즉시 결제자 데이터가 반영**되도록 변경했습니다.
- UI 전체 폰트를 **Paperlogy(Paperozi)** 계열로 적용했습니다.
- 로그인 화면과 상단 바에 **로고를 추가**했습니다.
- 화면 내 안내 문구를 줄이고, 노출 메시지를 전반적으로 공손한 표현으로 정리했습니다.

## 상세 변경 파일
### 1) `index.html`
- 로그인 영역을 참조 파일 스타일에 맞게 재구성했습니다.
- `currentSourceLabel` 영역을 제거했습니다.
- `syncPayersBtn` 버튼을 제거했습니다.
- 로그인/상단 헤더에 로고 이미지 경로를 연결했습니다.

### 2) `assets/classaround-logo.png`
- 업로드해 주신 참조 ZIP의 로고를 프로젝트 내부로 복사했습니다.

### 3) `src/css/styles.css`
- 로그인 화면 스타일을 참조 UI 방향으로 재정리했습니다.
- Paperlogy 웹폰트를 적용했습니다.
- 카드/버튼/헤더 스타일을 참조 UI 톤에 맞춰 다듬었습니다.

### 4) `src/js/auth.js`
- Google GIS 기본 렌더 버튼 대신 **커스텀 버튼 1개**로 로그인하도록 변경했습니다.
- 로그인 후 사용자 프로필을 가져와 상단에 표시하도록 정리했습니다.

### 5) `src/js/app.js`
- 시트 선택 변경 시 `syncPayersForActiveCoach()`가 즉시 실행되도록 수정했습니다.
- 로그인 직후 기본 시트가 자동 반영되도록 처리했습니다.
- 기존 동기화 버튼 이벤트를 제거했습니다.

### 6) `src/js/render.js`
- 시트 링크 라벨 관련 렌더링을 제거했습니다.
- 상태 문구를 현재 UI에 맞게 정리했습니다.

### 7) `src/js/config.js`
- 앱 타이틀을 `정산 매칭 보드`로 정리했습니다.
- 로그인 후 사용자 정보 조회를 위해 OAuth scope에 `openid email profile`을 추가했습니다.

## 참고
- 결제자 원본은 계속 **고정 스프레드시트 URL**을 사용합니다.
- 신청자 명단 업로드는 이번 수정에서도 유지했습니다.
